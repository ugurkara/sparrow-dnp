# Sparrow DNP

Sparrow-DNP provides wrapper and adapter classes for the java library of the [Opendnp3](https://github.com/dnp3/opendnp3) in order to configure by XML format. 

### Requirements

  * [Opendnp3](https://github.com/dnp3/opendnp3)
  * Maven
  * JDK 9 or above
